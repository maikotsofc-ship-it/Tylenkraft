# 📖 Sobre

**TylenKraft** é um jogo sandbox em desenvolvimento inspirado nos clássicos jogos de sobrevivência, mineração, exploração e construção.

O objetivo é criar um mundo vivo, cheio de possibilidades, permitindo que o jogador explore, colete recursos, construa estruturas e evolua ao longo da aventura.

Todo o projeto está sendo desenvolvido do zero em **Java**, com foco em desempenho, organização do código e compatibilidade com dispositivos Android.

---

# ✨ Recursos Planejados

- 🌍 Mundo gerado proceduralmente
- ⛏️ Mineração
- 🌳 Coleta de recursos
- 🏠 Construção de estruturas
- 🎒 Inventário
- 📦 Sistema de itens
- 🔥 Sistema de crafting
- 🌞 Ciclo de dia e noite
- 🌧️ Clima dinâmico
- 👾 Criaturas
- ❤️ Sistema de vida
- 🍖 Sistema de fome
- ⚡ Otimizações para celulares
- 💾 Salvamento automático
- 🎵 Sons e músicas
- 📱 Interface otimizada para Android
